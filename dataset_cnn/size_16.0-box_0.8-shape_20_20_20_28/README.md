# Example voxel-grid pickle

The representative pickle file for this dataset configuration,
`FAU-methanol_240_water_960-hydrophilic-02_01_02_propanol-size_16.0-box_0.8-shape_20_20_20_28.pkl`,
is not stored in this repository because the serialized file is larger than
GitHub's 100 MB file-size limit (the example file is >400 MB).

Readers can regenerate the same pickle locally from the molecular simulation
inputs and Python preprocessing scripts included in this repository.

## Dataset configuration

- Grid size: `16.0 Å`
- Grid increment: `0.8 Å`
- Grid shape: `20 × 20 × 20 × 28`
- Channels:
  - `0–13`: adsorbate features
  - `14–27`: solvent features
- Example system:
  - Zeolite: `FAU`
  - Environment: `methanol_240_water_960-hydrophilic`
  - Adsorbate: `02_01_02_propanol`

Each pickle corresponds to one zeolite–environment–adsorbate system. Under the
manuscript defaults, the script processes snapshots `intE01` through `intE10`
and applies the 24 proper cube rotations, producing 240 augmented voxel grids
for model training.

## Generate the example pickle

From the repository root, run:

```bash
python python_scripts/store_grids_pickle.py \
  --test \
  --zeolite FAU \
  --environment methanol_240_water_960-hydrophilic \
  --adsorbate 02_01_02_propanol
```

If the pickle already exists and you want to rebuild it, add:

```bash
--force-regenerate
```


## Output location

Generated pickle files are written to:

```text
dataset_cnn/size_16.0-box_0.8-shape_20_20_20_28/
```

The example file generated by the command above will be:

```text
FAU-methanol_240_water_960-hydrophilic-02_01_02_propanol-size_16.0-box_0.8-shape_20_20_20_28.pkl
```
